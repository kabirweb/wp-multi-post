== Description ==
WP Multi Post is a wordpress plugin, which make your multi post creation experience easier and faster.

== Installation ==
1. Upload `wp-multi-post` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==
= How many posts we can create at a time? =
Up to ten(10) posts.
